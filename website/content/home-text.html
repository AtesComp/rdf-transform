Getting started...

  1. Download the RDF Transform extension
  2. Stop OpenRefine
  3. If not present, add an "extensions" directory exists in your OpenRefine workspace
  4. If not present, add an "rdf_transform" subdirectory to the "extensions" directory
  5. Extract the downloaded zip file to the "rdf_transform" subdirectory
  6. Restart OpenRefine
Visit the installation guide for more detailed instructions. For tutorials-like material see the showcases. For detailed and technical documentation, see the documentation section.
