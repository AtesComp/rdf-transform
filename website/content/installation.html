• Locate the OpenRefine workspace directory. You can locate it on the OpenRefine start page as seen in the figure below.
  • If the workspace does not contain a directory named "extensions", create it.
  • In the "extensions" directory, create an "rdf_transform" directory.
  • Download the RDF Transform extension.
  • Unzip the downloaded file to the "rdf_transform" folder you created in step 3.
  • Restart OpenRefine. You should see the "RDF" menu item in the upper right corner right of "Extensions:" as shown:
