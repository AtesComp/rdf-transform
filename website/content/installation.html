• Locate the OpenRefine workspace directory. You can locate it on the OpenRefine start page as seen in the figure below.
  • If the workspace does not contain a directory named "extensions", create it.
  • Enter the "extensions" directory.
  • Download the RDF Transform extension to the "extensions" directory.
  • Unzip the downloaded file. It should create an "rdf-transform" folder.
  • Restart OpenRefine. You should see the "RDF Transform" menu item in the upper right corner right of "Extensions:" as shown:
