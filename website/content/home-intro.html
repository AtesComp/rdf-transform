The RDF Transform Extension for OpenRefine enables exporting interlinked RDF data from OpenRefine projects.

Main Features:

  • RDF Transforms
    • A GUI for defining the transformation of data to an RDF graph
    • Control your RDF namespaces
    • Use your own vocabulary or import existing ones
    • Autocomplete for class and property names
    • Use GREL to fine tune your resources and literals
  • RDF Exports
    • Export your transformed data into RDF/XML or Turtle
  • Extendable
    • Install RDF Reconcile to reconcile terms with existing RDF ontologies
